# 📚 Đánh giá Nguồn thông tin Học thuật: Trực quan hóa Git DAG

## 📖 Giới thiệu (About The Project)
Đây là kho lưu trữ chứa Báo cáo đánh giá nguồn thông tin học thuật thuộc lĩnh vực Khoa học Máy tính. Báo cáo tập trung vào việc thu thập, phân tích và đánh giá độ tin cậy của các tài liệu liên quan đến **Phương pháp trực quan hóa cấu trúc dữ liệu Đồ thị có hướng không chu trình (DAG) và Lịch sử tiến hóa phần mềm trong hệ thống Git**.

Dự án này bao gồm việc sàng lọc và đánh giá chi tiết 13 nguồn tài liệu uy tín từ các tạp chí khoa học (IEEE Xplore), sách chuyên khảo và các blog kỹ thuật hàng đầu.

## 🚀 Nội dung chính (Key Contents)
- Báo cáo chi tiết định dạng HTML (được thiết kế chuẩn layout A4, font Times New Roman).
- Bảng tổng hợp và đánh giá độ tin cậy dựa trên 5 tiêu chí: tác giả, cơ quan xuất bản, phương pháp nghiên cứu, trích dẫn và tính cập nhật.
- Phân tích chuyên sâu về sự chuyển dịch từ các công cụ đếm dòng code (LOC) sang phân tích Cây cú pháp trừu tượng (AST) và thuật toán phân cụm.
- Danh mục tài liệu tham khảo theo chuẩn Harvard.

## 🛠️ Hướng dẫn xem báo cáo (How to View)
Báo cáo được viết dưới dạng mã HTML thuần để giữ nguyên định dạng cấu trúc học thuật. Để xem báo cáo:
1. Clone repository này về máy:
   ```bash
   git clone https://github.com/ThanhNguyn/git-visualization-research
   ```
2. Mở file `index.html` (hoặc tên file bạn đã lưu) bằng bất kỳ trình duyệt web nào (Chrome, Edge, Firefox, Safari).
3. Hoặc có thể copy trực tiếp nội dung hiển thị trên trình duyệt dán vào Microsoft Word để in ấn/nộp bài.
